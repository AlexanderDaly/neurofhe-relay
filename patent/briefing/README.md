## Framing Principle

Use the phrase "privacy-preserving neural computation infrastructure." Avoid public-facing language that implies generalized cognition decoding, surveillance capability, or medical efficacy. The central ENER message is narrower and more defensible: raw neural telemetry should remain local when possible, selected latent representations should be minimized before encryption, and external compute should operate on protected embeddings rather than plaintext neural signals.
